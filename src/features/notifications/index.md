# Notifications #
Used to notify user about something.
There are several notification options:

| type  | description                                                   |
|-------|---------------------------------------------------------------|
| push  | notification that appears even if app closed or in background |
| toast | temporary or permanent toast that appears over the UI         |
| popup | some internal popup                                           |

## Remote ##
For remote notifications we use Firebase
